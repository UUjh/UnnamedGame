# UnnamedGame

게임 이름: Whale is so Cute.

플랫폼: Unity/Window

장르: 아케이드, 어드벤처

- 소개


꼬리와 지느러미가 작아 이동 속도가 느리며,
앞으로 나아가기 위해서는 누구보다 빠르게 꼬리와 
지느러미를 움직여야 한다. 하지만 천성이 게을러 멸종 위기에 처해 있다.

고래를 조종해 우리의 푸른 고래를 보호하고, 바다의 왕으로 만들자.

- 장애물

푸른 고래의 주식 물고기 트리오

푸른 고래가 언제나 즐겨 먹는 물고기 트리오. 
속도가 느리고, 매사에 무관심 해 푸른 고래가 쫓아와도 모른다. 
 

푸른 고래의 영원한 숙적 거대상어

 푸른 고래가 성체가 되기 전까지는 푸른 고래를 주식으로 먹는다. 푸른 고래가 가까이 있으면 빠르게 쫓아와 잡아먹는다.
 하지만 푸른 고래가 성체가 되면 푸른 고래의 고단백 영양 
식사가 된다.


인간의 욕심은 끝이 없고 똑같은 실수를 반복하지 보트

큰 소리를 내며 바닷속 친구들을 괴롭게 만드는 층간소음의 주범. 
어린 푸른 고래는 연약해 보트에 부딪히면 죽지만, 푸른 고래가 성체가 되면 참지 않고 보트를 부술 수 있다.

 
- 게임 방법


이동 방법
X, C, V, B를 연타해 이동 속도를 높일 수 있다.
방향키로 시점을 전환할 수 있다.

생존 방법
물 위로 올라가 산소를 채울 수 있고, 물고기를 먹어 체력을 채울 수 있다.

성체가 되기 전까지는 상어를 피해 다녀야 한다. 상어가 쫓아온다면 뒤도 돌아보지 말고 도망쳐라.

바닷속 왕이 되기 전까지 보트를 부술 수 없다. 보트가 보인다면 물 밑으로 들어가 보트를 피해라.
